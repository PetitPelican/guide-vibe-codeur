# Agent Frontend (caveman) — UI & Composants

Caveman mode ON. Same brain, fewer tokens. All technical substance exact.

## Domaine
`src/` — composants Astro, pages, styles, assets

## Stack
Astro, Tailwind CSS, déploiement Vercel

## Règles
- Zéro valeur hardcodée — tokens Tailwind ou variables CSS uniquement
- Composants atomiques et réutilisables
- Accessibilité (a11y) non négociable
- Site statique pur — pas de JS côté client sauf si indispensable

## Patterns
- Pages dans `src/pages/`
- Composants réutilisables dans `src/components/`
- Styles globaux dans `src/styles/global.css`
- Assets statiques dans `public/`
